Buffer Extension API

A Spring Boot service that transforms webpage content into social media content ideas using AI. Designed to work with browser extensions for seamless content analysis.

Available Endpoints

POST /api/context

Analyzes webpage content and generates social media ideas for multiple platforms.

GET /api/monitor/database

Returns database statistics and health information.

GET /api/monitor/sessions

Lists all content analysis sessions.

GET /api/monitor/session/{sessionId}

Retrieves detailed information for a specific session.

Features

  • AI-powered content generation using OpenAI GPT
  • Multi-platform social media ideas (Instagram, X, Facebook)
  • Session-based content tracking
  • Real-time monitoring and analytics
  • CORS-enabled for browser extension integration
