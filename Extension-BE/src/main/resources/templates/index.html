Buffer Extension API

Welcome to the Buffer Extension API service. This service provides an endpoint for processing URLs and selected text using OpenAI's GPT model.

API Endpoint

POST /api/chat

Request Body

{
    "url": "https://example.com",
    "selectedText": "Text to analyze",
    "chatId": "" // Optional, will be generated if not provided
}

Response

{
    "chatId": "generated-chat-id",
    "response": "OpenAI's response"
}

Example Usage

Using curl:

curl -X POST http://localhost:8080/api/chat \
-H "Content-Type: application/json" \
-d '{
    "url": "https://example.com",
    "selectedText": "This is a sample text to analyze",
    "chatId": ""
}'

Features

  • Process URLs and selected text
  • Maintain conversation history using chatId
  • Powered by OpenAI's GPT model
  • RESTful API design

Notes

  • The service maintains chat history for each conversation using the chatId
  • If no chatId is provided, a new one will be generated
  • To continue a conversation, include the chatId from a previous response
